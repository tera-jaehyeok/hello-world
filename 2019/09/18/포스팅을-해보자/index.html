포스팅을 해보자

9월 18 2019 1분 소요 (약 143 단어)
  • 사전 설치
    • nodejs (hexo 가 nodejs 기반)
  • 시작하기
    • 프로젝트를 다운 받는다. github
    • npm install 을 한다. (보통 인텔리j 에서 설치 알림 팝업이 생성됨)
    • $ hexo s 입력하여 로컬 서버에 실행한다
    • http://localhost:4000/admin 으로 들어간다
    • 계정은 techman 계정과 동일하다 (ga 에서 사용중)
    • Posts 에서 신규 글 및 기존 글 수정이 가능하다
    • Deploy 에서 Deploy 버튼으로 배포
    • 변경된 소스를 github 에 merge 한다.
